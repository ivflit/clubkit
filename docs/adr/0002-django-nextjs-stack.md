# Django backend with Next.js frontend

We use Django + Django REST Framework as the backend and Next.js + React + Tailwind CSS as the frontend. Django was chosen over a Node.js backend because the developer is proficient in Python/Django, and Django provides first-class support for the project's core needs: a built-in ORM with PostgreSQL schema routing (critical for separate-schema multi-tenancy), a mature auth system, and a migration framework that can be scripted to run across all Tenant schemas. The Next.js frontend handles presentation, server-side rendering for SEO on public club pages, and all admin dashboards (Platform Admin and Tenant Admin). Django's built-in admin panel is not used — all admin interfaces are custom-built in the Next.js frontend for a modern UX.
